## Tool Acquisition Sources

### New Tools (Quality)
- **Park Tool** - Industry standard, lifetime warranty
- **Pedro's** - Quality, good value
- **Unior** - European, professional grade
- **Feedback Sports** - Repair stands, workstations

### New Tools (Budget)
- **Amazon Basics** - Hit or miss quality
- **Bikehand** - Decent budget option
- **Local hardware stores** - General tools

### Used/Donated Tools
- Bike shop closeouts
- Estate sales
- Tool drives
- Corporate donations
- Grants (many fund equipment)

---

## Tool Maintenance

### Regular Maintenance
- Wipe down tools after use
- Oil moving parts periodically
- Sharpen cutting tools as needed
- Replace worn grips/handles
- Calibrate torque wrenches annually

### Storage
- Keep tools dry
- Hang when possible (pegboard)
- Organize by function
- Label storage locations
- Inventory regularly

## Tool Check-Out System

### For Volunteer Shops
1. Tool crib with check-out sheet
2. Sign tools in/out by name
3. Count tools at shift end
4. Report missing/damaged immediately
