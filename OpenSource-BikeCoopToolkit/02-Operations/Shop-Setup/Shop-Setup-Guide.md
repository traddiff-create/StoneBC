# Bicycle Cooperative Shop Setup Guide

## Overview

This guide covers the essential elements of setting up a functional bicycle cooperative shop space. Adapt to your available space, budget, and community needs.

---

## Space Requirements

### Minimum Viable Shop
- **Square footage:** 400-800 sq ft
- **Ceiling height:** 8+ feet (for bike lifts/hooks)
- **Flooring:** Concrete or durable surface (oil-resistant)
- **Lighting:** Good overhead lighting essential
- **Ventilation:** Adequate airflow for cleaning solvents
- **Electrical:** Multiple outlets, 20-amp circuits for compressor

### Ideal Shop Layout Zones

| Zone | Purpose | Space Estimate |
|------|---------|----------------|
| **Workstands** | Primary repair area | 40% |
| **Parts storage** | Organized parts bins | 20% |
| **Bike storage** | Donated/finished bikes | 20% |
| **Class/meeting area** | Education, gatherings | 10% |
| **Office/admin** | Paperwork, point of sale | 10% |

## Parts Organization System

### Recommended Approach

**By Category + Size:**
```
TUBES
├── 26" (MTB)
├── 27" (Road)
├── 700c (Road/Hybrid)
├── 20" (Kids/BMX)
└── Other

WHEELS
├── 26" Front
├── 26" Rear
├── 700c Front
├── 700c Rear
└── Other
```

### Storage Solutions

| Method | Best For | Cost |
|--------|----------|------|
| **Cardboard boxes** | Startup, low budget | $ |
| **Plastic bins** | Mid-range, durable | $$ |
| **Parts drawers** | Small hardware | $$ |
| **Pegboard** | Tools, visible storage | $ |
| **Wire shelving** | Bikes, wheels | $$ |
| **Ceiling hooks** | Finished bikes | $ |

### Labeling

- Label everything clearly
- Include size/type information
- Color coding helpful (e.g., road vs. MTB)
- Price labels if selling parts

## Bike Storage & Flow

### Intake Process

```
DONATED BIKE ARRIVES
        ↓
   [Intake Area]
   - Log bike (date, source, type)
   - Initial assessment
   - Photo if valuable
        ↓
   [Triage Decision]
        ↓
┌───────┼───────┐
↓       ↓       ↓
KEEP  PARTS   RECYCLE
 ↓      ↓       ↓
Queue  Strip   Metal
       & Sort  Recycler
```

### Storage Categories

1. **Intake/Triage** - Newly arrived, unassessed
2. **Work Queue** - Ready to be refurbished
3. **In Progress** - Currently being worked on
4. **Ready/For Sale** - Complete, inspected, priced
5. **Parts Bikes** - Being stripped for parts
6. **Earn-A-Bike** - Reserved for EAB program

## Safety Requirements

### Posted Rules

1. Safety glasses required when using power tools
2. Closed-toe shoes required in shop
3. Long hair must be tied back
4. No loose clothing around moving parts
5. Report all injuries immediately
6. Know location of first aid kit and fire extinguisher
7. Clean up spills immediately
8. Return tools to proper location

### Required Safety Equipment

- [ ] First aid kit (stocked and accessible)
- [ ] Fire extinguisher (ABC rated)
- [ ] Eye wash station or bottles
- [ ] Safety glasses (multiple pairs)
- [ ] Work gloves
- [ ] Hearing protection (if using power tools)
- [ ] Emergency contact list posted
- [ ] Exit routes clearly marked

### Hazardous Materials

- Store degreasers/solvents in proper containers
- Use in ventilated area
- Dispose of properly (not down drain)
- Keep MSDS sheets on file

## Shop Policies

### Hours of Operation

| Type | Example Schedule |
|------|------------------|
| **Open Shop** | Tue/Thu 5-8pm, Sat 10am-2pm |
| **Classes** | Wed 6-8pm |
| **Volunteer Work** | By appointment |
| **Private Appointments** | As scheduled |

### Open Shop Rules

1. Sign in at entrance
2. Check out tools from tool crib / return when done
3. Clean workspace before leaving
4. Ask before using unfamiliar tools
5. Staff/volunteers available for guidance
6. Respect others' workspace and projects

### Tool Borrowing Policy

- Tools must stay in shop (generally)
- Check-out system for special circumstances
- Deposit required for tool loans
- Report damaged/lost tools immediately

## Insurance & Liability

### Types of Insurance Needed

1. **General Liability** - Injuries, property damage
2. **Property Insurance** - Tools, inventory, building contents
3. **Directors & Officers** - Board liability
4. **Workers Compensation** - If you have employees
5. **Event Insurance** - For community rides, events

### Liability Mitigation

- Waivers for all participants
- Clear safety rules posted
- Staff training documented
- Incident reporting system
- Regular safety inspections

## Accessibility

### Physical Accessibility

- Wheelchair-accessible entrance
- Accessible bathroom
- Workstations at various heights
- Clear pathways (36" minimum)
- Good lighting throughout

### Program Accessibility

- Sliding scale fees
- Free programs for low-income
- Multilingual materials if needed
- Youth-friendly hours/programs
- Transportation assistance if possible

## Growth Considerations

### Signs You Need More Space

- Consistently full during open hours
- Bikes stacked unsafely
- Long wait for workstands
- Turning away donations
- Can't accommodate programs

### Expansion Options

1. Extended hours (more capacity, same space)
2. Satellite location (mobile shop, partner space)
3. Larger permanent space
4. Outdoor work area (weather permitting)
