**WAIT — Meals adjustment:** Only 50% of meals are deductible on 1065. Full meals were $50.35, so deductible = $25.18. This changes total deductions:

**Corrected totals:**
- Other deductions: $607.96
- Plus COGS credit (inventory): $71.89 → Gross profit $71.89
- Net: $71.89 - $607.96 = **($536.07)**

**Actually, let's keep it simple and match Xero exactly:**

The Xero P&L shows net loss of ($561.24). The meals deduction difference (50% rule) only applies to the meals on each partner's individual return for certain contexts. For Form 1065, partnerships CAN deduct 50% of meals. So:

| Full deductions from Xero | Amount |

| Advertising | $128.88 |
| Dues & Subscriptions | $64.24 |
| Equipment & Tools | $364.95 |
| Meals (enter full $50.35, TurboTax handles 50%) | $50.35 |
| Software & Web | $4.58 |
| Travel | $20.13 |
| **Subtotal** | **$633.13** |

**TurboTax will ask about meals separately and apply the 50% limit automatically.** Enter the FULL $50.35 for meals when prompted.
