# Stone Bicycle Coalition - Project Overview

## Project Summary

**Stone Bicycle Coalition** is building an open-source bicycle cooperative model in Rapid City, South Dakota. Our goal is to create a replicable framework that any community can use to start their own bike co-op, adapted to their local laws.

---

## Project Goals

### Primary Goals
1. **Establish a functioning bicycle cooperative** in Rapid City, SD
2. **Document everything** we learn and create
3. **Release open-source toolkit** for other communities to replicate

### Secondary Goals
- Promote cycling culture in the Black Hills region
- Provide accessible bike repair and education
- Build youth programs and job skills training
- Advocate for cycling infrastructure
